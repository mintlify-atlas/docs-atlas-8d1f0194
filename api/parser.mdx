title: 'Parser'
description: 'Parse Yoyo source code into abstract syntax trees'
---

The `Parser` class transforms Yoyo source code into an abstract syntax tree (AST) that can be compiled and executed by the engine.

## Class Definition

```cpp
class YOYO_API Parser
{
public:
    explicit Parser(std::string& source);
    std::unique_ptr<ASTNode> parseExpressionOrDeclaration();
    std::unique_ptr<Expression> parseExpression(uint32_t precedence);
    std::unique_ptr<Statement> parseStatement();
    std::vector<std::unique_ptr<Statement>> parseProgram();
    [[nodiscard]] bool failed() const;
    // ... additional methods
};
```

## Constructor

### Parser()

Creates a parser instance for the given source code.

```cpp
explicit Parser(std::string& source);
```

#### Parameters

<ParamField path="source" type="std::string&" required>
  Reference to the Yoyo source code string. The reference must remain valid for the lifetime of the parser.
</ParamField>

#### Example

```cpp
#include <yoyo/parser.h>

std::string code = "fn add(a: i32, b: i32) -> i32 { return a + b; }";
Yoyo::Parser parser(code);
```

## Primary Parsing Methods

### parseProgram()

Parses an entire Yoyo program, returning a vector of top-level statements.

```cpp
std::vector<std::unique_ptr<Statement>> parseProgram();
```

#### Returns

<ResponseField name="return" type="std::vector<std::unique_ptr<Statement>>">
  Vector of parsed top-level statements representing the complete program
</ResponseField>

<Note>
After calling this method, always check `failed()` to ensure parsing succeeded before using the results.
</Note>

#### Example

```cpp
#include <yoyo/parser.h>
#include <iostream>

std::string source = R"(
    fn factorial(n: i32) -> i32 {
        if n <= 1 { return 1; }
        return n * factorial(n - 1);
    }
)";

Yoyo::Parser parser(source);
auto statements = parser.parseProgram();

if (parser.failed()) {
    std::cerr << "Parse error occurred" << std::endl;
} else {
    std::cout << "Successfully parsed " << statements.size() << " statements" << std::endl;
}
```

---

### parseExpression()

Parses a single expression with the specified precedence level.

```cpp
std::unique_ptr<Expression> parseExpression(uint32_t precedence);
```

#### Parameters

<ParamField path="precedence" type="uint32_t" required>
  Minimum precedence level for operator parsing. Use 0 for lowest precedence (parse entire expression).
</ParamField>

#### Returns

<ResponseField name="return" type="std::unique_ptr<Expression>">
  Parsed expression AST node, or nullptr if parsing failed
</ResponseField>

#### Example

```cpp
std::string expr = "2 + 3 * 4";
Yoyo::Parser parser(expr);
auto expression = parser.parseExpression(0);

if (!parser.failed()) {
    // Expression successfully parsed
    // Will respect operator precedence: 2 + (3 * 4)
}
```

---

### parseExpressionOrDeclaration()

Parses either an expression or a declaration, automatically detecting which one is present.

```cpp
std::unique_ptr<ASTNode> parseExpressionOrDeclaration();
```

#### Returns

<ResponseField name="return" type="std::unique_ptr<ASTNode>">
  Parsed AST node (either Expression or Statement/Declaration)
</ResponseField>

#### Example

```cpp
std::string code1 = "let x = 10;";
std::string code2 = "x + 5";

Yoyo::Parser parser1(code1);
auto node1 = parser1.parseExpressionOrDeclaration();  // Declaration

Yoyo::Parser parser2(code2);
auto node2 = parser2.parseExpressionOrDeclaration();  // Expression
```

## Error Handling

### failed()

Checks if any parse errors occurred.

```cpp
[[nodiscard]] bool failed() const;
```

#### Returns

<ResponseField name="return" type="bool">
  `true` if parsing encountered errors, `false` otherwise
</ResponseField>

<Note>
The `[[nodiscard]]` attribute means the compiler will warn if you don't check the return value.
</Note>

#### Example

```cpp
Yoyo::Parser parser(source);
auto program = parser.parseProgram();

if (parser.failed()) {
    std::cerr << "Parsing failed!" << std::endl;
    return 1;
}

// Safe to use program
```

---

### error()

<ParamField path="tk" type="std::optional<Token>" required>

## Complete Usage Example

```cpp
#include <yoyo/parser.h>
#include <yoyo/engine.h>
#include <iostream>
#include <fstream>
#include <sstream>

std::string loadFile(const std::string& filename) {
    std::ifstream file(filename);
    std::stringstream buffer;
    buffer << file.rdbuf();
    return buffer.str();
}

int main() {
    // Load Yoyo source file
    std::string source = loadFile("example.yy");
    
    // Create parser
    Yoyo::Parser parser(source);
    
    // Parse the entire program
    auto statements = parser.parseProgram();
    
    // Check for errors
    if (parser.failed()) {
        std::cerr << "Failed to parse program" << std::endl;
        return 1;
    }
    
    std::cout << "Successfully parsed " << statements.size() 
              << " top-level statements" << std::endl;
    
    // You can now pass these statements to the Engine for execution
    // MyEngine engine;
    // engine.sources["main"] = {source, std::move(statements)};
    // engine.execute();
    
    return 0;
}
```
